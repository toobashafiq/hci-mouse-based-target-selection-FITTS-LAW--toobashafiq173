 Fitts' Law Experiment - Mouse Based Target Selection

 🎯 Features

- **Start Block & Target Selection**  
  - Click on the START circle to begin a trial.  
  - Move the mouse to reach the randomly generated TARGET.

- **Multiple Difficulty Levels**  
  - Normal, Medium, Hard (different numbers of obstacles).

- **Obstacles and Ghosts**  
  - Random obstacles appear depending on the level.  
  - Ghosts move dynamically to increase task difficulty.

- **Fitts’ Law Metrics**  
  - Straight distance (D)  
  - Target width (W)  
  - Index of Difficulty (ID) = log2(D/W + 1)  
  - Movement Time (MT) for each trial  

- **Path Tracking**  
  - Mouse path is traced for each trial.  
  - Distance along the path is calculated.

- **Trial Logging and CSV Export**  
  - Logs trial number, distances, ID, MT, hit/miss, and timestamp.  
  - Download experiment data as CSV for analysis.
